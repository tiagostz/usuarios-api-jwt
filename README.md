# 🚀 Projeto Full Stack - Autenticação com JWT

Este projeto foi desenvolvido com foco em boas práticas de desenvolvimento, priorizando segurança, organização e experiência do usuário.

---

## 🔐 Segurança
- Criptografia de senhas utilizando hashing
- Autenticação baseada em JWT (JSON Web Token)
- Controle de sessões e acesso a rotas protegidas

---

## 🧩 Arquitetura
- Separação entre rotas públicas e privadas
- Uso de middlewares para autenticação
- Estrutura organizada e modular no backend

---

## 💻 Frontend
- Desenvolvido com **React**
- Estilização com **Tailwind CSS**
- Interface limpa, moderna e responsiva

---

## 🔗 Integração
- Comunicação entre frontend e backend via **Axios**
- Persistência de autenticação utilizando **localStorage**

---

## 🗄️ Banco de Dados
- Utilização do **MongoDB**
- Armazenamento e gerenciamento eficiente dos dados

---

## ⚙️ Tecnologias Utilizadas

### Backend
- Node.js
- Express
- MongoDB
- JWT

### Frontend
- React
- Tailwind CSS
